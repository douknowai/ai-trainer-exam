第一章　系统简介

1.1 产品定位

本系统是面向人工智能训练师五级职业资格认证的在线练习与考试平台，服务于职业培训学校的教学全流程。系统覆盖从题库管理、日常练习、实操训练到正式考试、成绩发布、数据分析的完整业务链路。

1.2 系统角色与职责

角色标识核心职责
学员student参加理论练习、实操任务、正式考试，查看成绩与错题
教师teacher查看班级学员情况，管理考试安排，跟进学习进度
超级管理员super_admin管理用户/组织/班级/题库/考试/成绩/系统配置
学校管理员school_admin本组织范围内的管理功能
题库编辑question_editor创建和编辑题目，提交审核
题库审核question_reviewer审核题目，批准或驳回
监考员invigilator监控考试进行状态
审计员auditor查看审计日志

1.4 设计理念

系统遵循"零基础友好"原则：

  • 大字号、大按钮、大点击区域：学员正文 18px，主按钮文字 18px，最小正文 14px
  • 每步操作有反馈：答题后立即显示"做对了！"或"答错了，没关系"等温和提示
  • 状态绝不只用颜色表达：必须图标 + 文字双重标识
  • 练习温和、考试严肃：两种模式视觉风格截然不同
  • 禁止专业术语直出：错误提示必须说明"哪里错、怎么改"
提示系统配色以墨青绿为主色（源自教室护眼黑板），暖白纸为背景，营造稳重可信赖的氛围。

第三章　快速开始

3.1 访问系统

在浏览器地址栏中输入系统访问地址，按回车键进入登录页面。建议将地址添加到浏览器书签。

3.2 登录与退出

登录

  1. 在登录页面输入邮箱地址和密码
  2. 点击"登录"按钮
  3. 系统验证成功后，自动跳转到对应角色的工作台
注意连续多次输入错误密码可能导致账号被临时锁定。如忘记密码，请联系系统管理员重置。

退出

  1. 点击页面右上角的用户头像/姓名
  2. 选择"退出登录"
  3. 系统返回登录页面
提示系统默认在无操作 120 分钟后自动登出（管理员可在系统设置中调整）。考试期间不受此限制。

第四章　学员操作指南

学员登录后进入学员端，顶部导航栏提供以下功能入口：

入口功能说明

4.1 学员首页

  • 欢迎信息：显示学员姓名
  • 练习统计：累计练习次数、正确率百分比
  • 考试动态：当前可参加的考试列表及状态
  • 快捷入口：一键跳转到练习或考试

4.2 理论练习

操作路径：学员端 -> 理论练习

4.2.1 开始练习

  1. 进入理论练习页面，系统自动从练习题库中加载题目
  2. 题型包括：单选题（四个选项 A/B/C/D）和判断题（对/错）
  3. 每道题展示：题目内容、选项、难度等级

4.2.2 答题与反馈

答题结果显示效果说明
答对绿色对勾图标 + "做对了！"文字鼓励性反馈，增强信心
答错红色叉号图标 + "答错了，没关系" + 正确答案温和提示，展示正确答案

4.2.3 练习流程

加载题目->阅读题干->选择答案->即时反馈->下一题

4.2.4 错题自动收录

答错的题目会自动加入"错题本"，记录错题次数。同一道题答对后，将不再出现在错题列表中。

4.3 实操任务（7 种题型）

操作路径：学员端 -> 实操任务

实操任务模拟真实的人工智能训练师工作场景，包含以下 7 种任务类型：

任务类型操作说明考核技能
Excel 删除行查看数据表格，找出需要删除的异常行并标记数据清洗、表格操作
统计表填写根据源数据计算统计值并填入对应单元格数据统计、计算能力
文件分类将文件按类型（图片/文档/音频等）拖拽归类文件管理、分类判断
图片清洗从一组图片中选出不符合要求的样本（如模糊、遮挡）图片质量判断
图片标注在图片中的目标对象上画框并标注类别目标检测标注
情感标注阅读文本，判断情感倾向（正面/负面/中性）文本理解、情感分析
音频转写播放音频，将听到的内容转写为文字语音识别辅助、文字录入

4.3.1 通用操作流程

查看任务列表->点击"开始"->阅读任务目标->完成操作->点击"提交"->查看评分

4.3.2 任务界面布局

每个实操任务页面采用统一的"三段式"布局：

  • 顶部：一句话任务目标说明（20px 加粗）
  • 中部：大操作区，根据任务类型展示不同的交互界面
  • 底部：固定的主操作栏，包含"提交"按钮
提示实操任务页面专为零基础学员设计：大字号、大按钮、每步有反馈。建议先完成几道练习熟悉操作方式。

4.4 正式考试（完整流程）

操作路径：学员端 -> 正式考试

4.4.1 考试完整流程

查看考试列表->进入考试->逐题作答->交卷->查看成绩

4.4.2 考试状态说明

状态含义可操作
未开始当前时间早于考试开始时间否

4.4.3 答题界面

  • 倒计时：页面顶部显示剩余时间，精确到秒
  • 题目导航：可随时切换上一题/下一题，已答和未答有不同标记
  • 交卷按钮：位于答题区底部，点击后确认提交
重要倒计时结束后系统会自动交卷。请合理安排答题时间，优先完成有把握的题目。

4.4.4 考试规则

规则项说明

4.5 成绩查询

4.6 错题本

学习建议建议在正式考试前，先过一遍错题本中的题目，重点复习高频错题。

第五章　教师操作指南

教师登录后进入教师端，主要功能包括：仪表盘、考试管理、学员管理。

5.1 教师仪表盘

  • 班级概览：当前教师所带班级的学员总数、近期活跃情况
  • 考试概览：已安排的考试列表、参与人数、平均分趋势

  • 查看学员练习统计：累计练习次数、正确率
  • 查看学员考试记录：历次考试分数、通过状态

5.4 教学建议

场景建议操作

6.6 成绩管理与复核流程

6.6.1 成绩状态流转

自动评分->人工复核->发布成绩
状态含义可操作

已复核管理员已确认/调整分数可发布
已发布成绩已对学员可见不可修改

6.6.2 成绩复核操作

  1. 在成绩列表中点击"复核"按钮
  2. 查看每道题的作答详情和评分结果
  3. 如需调整分数，输入调整后的分数
  4. 标记通过/不通过，填写复核备注
  5. 确认提交

6.6.3 批量发布成绩

  1. 在成绩列表中勾选需要发布的学生
  2. 点击"批量发布"按钮
  3. 确认发布（发布后学员端可见成绩）
注意成绩发布后无法撤回。请确保所有复核工作已完成后再发布。

第八章　故障排查与常见问题

Q1: 登录提示"邮箱或密码错误"怎么办？

请检查邮箱地址是否正确（注意大小写），密码是否正确。如忘记密码，请联系系统管理员重置。

Q2: 考试时突然断网了，答题内容会丢失吗？

不会。系统支持断网恢复。网络恢复后重新登录进入考试，之前已答的题目会保留。系统会在服务端记录您的答题时间，不会因断网而判超时。

Q3: 倒计时结束后我还没答完怎么办？

系统会在倒计时结束后自动交卷，按您已完成的答题内容进行评分。建议合理安排时间，优先答有把握的题目。

Q4: 成绩显示"等待发布"，什么时候能看到分数？

管理员需要对成绩进行复核后才能发布。发布后您会在成绩查询页面看到具体分数。一般复核需 1-3 个工作日。

Q5: 同一道考试题可以重考吗？

每位学员对每场考试仅有一次答题机会。如因特殊原因需要补考，请联系教师或管理员重新安排。

Q6: 页面打开是空白或显示异常怎么办？

  1. 清除浏览器缓存（Ctrl+Shift+Delete）
  2. 关闭浏览器后重新打开
  3. 尝试使用无痕/隐私模式访问
  4. 更换浏览器（推荐 Chrome 或 Edge）
  5. 如仍有问题，请联系管理员并提供截图

Q7: 系统支持手机使用吗？

手机端可查看首页、成绩和进度。实操考试请在 PC 上进行，以保证最佳操作体验。

Q8: 如何批量导入题目？

管理员可通过 DOCX 格式批量导入题目。请按系统要求的模板格式编写文档，在题库管理 -> 导入题库页面上传。每道题需包含题干、选项、正确答案和知识点标签。

Q9: 考试迟到了还能进入吗？

在考试开始后的迟到入场时间内（默认 15 分钟）仍可进入考试。超过该时间将无法入场。具体时间以管理员设置为准。

Q10: 管理员如何发布成绩？

  1. 进入管理端 -> 成绩管理
  2. 复核完所有学员成绩
  3. 勾选需要发布的学员记录
  4. 点击"批量发布"按钮
  5. 确认发布（注意：发布后不可撤回）
