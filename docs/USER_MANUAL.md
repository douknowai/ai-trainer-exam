## 1. 系统简介

本系统是面向**人工智能训练师五级职业资格认证**的在线练习与考试平台，服务于职业培训学校的教学全流程。

### 1.1 系统角色

| 角色 | 职责 | 入口 |
|------|------|------|
| 学员 | 参加理论练习、实操任务、正式考试，查看成绩 | 学员端 |
| 教师 | 查看班级学员情况，管理考试安排 | 教师端 |
| 超级管理员 | 管理用户/组织/班级/题库/考试/成绩/系统配置 | 管理端 |

### 1.2 技术架构

- 前端：Next.js 16 + React 19 + TypeScript + shadcn/ui + Tailwind CSS 4
- 后端：Next.js API Routes (App Router)
- 数据库：Supabase PostgreSQL
- 认证：Supabase Auth (邮箱/密码)

### 1.3 设计理念

系统遵循**"零基础友好"**原则：

- **大字号、大按钮**：学员正文 18px，主按钮文字 18px，最小正文 14px
- **每步操作有反馈**：答题后立即显示"做对了！"或"答错了，没关系"
- **状态必须图标+文字**：不依赖颜色传达关键信息
- **练习温和、考试严肃**：两种模式视觉风格不同

## 3. 学员操作指南

学员登录后进入学员端，顶部导航栏包含以下功能入口：

- **首页** — 查看学习概览
- **理论练习** — 刷题练习
- **实操任务** — 动手操作练习
- **正式考试** — 参加考试
- **成绩查询** — 查看历次成绩
- **错题本** — 复习错题
- **帮助** — 使用帮助

### 3.1 学员首页

学员首页展示以下信息：

- **欢迎信息**：显示学员姓名
- **练习统计**：累计练习次数、正确率
- **考试动态**：当前可参加的考试、近期考试安排
- **快捷入口**：一键跳转到练习/考试

### 3.2 理论练习

**操作路径：** 学员端 → 理论练习

#### 开始练习

1. 进入理论练习页面，系统会自动加载练习题目
2. 题型包括：**单选题**和**判断题**
3. 每道题展示：题目内容、四个选项（或对/错）

#### 答题与反馈

1. 点击您认为正确的选项
2. 系统立即反馈：
   - **答对了！** — 显示绿色对勾图标 + "做对了！"
   - **答错了，没关系** — 显示红色叉号图标 + 正确答案
3. 点击"下一题"继续练习

#### 错题自动收录

- 答错的题目会自动加入"错题本"
- 您可以随时在错题本中回顾和重做

### 3.3 实操任务

**操作路径：** 学员端 → 实操任务

实操任务模拟真实的人工智能训练师工作场景，包含以下题型：

| 任务类型 | 说明 |
|----------|------|
| Excel 删除行 | 找出数据表中需要删除的行 |
| 统计表填写 | 根据数据填写统计表 |
| 文件分类 | 将文件按类型归类 |
| 图片清洗 | 从图片中选出不符合要求的样本 |
| 图片标注 | 为图片中的目标对象画框标注 |
| 情感标注 | 判断文本的情感倾向 |
| 音频转写 | 听音频并转写为文字 |

#### 操作流程

1. 进入实操任务页面，查看当前分配的任务列表
2. 点击"开始"进入任务详情
3. 阅读顶部的**任务目标**（一句话说明要做什么）
4. 在操作区域完成任务（如点击选项、勾选文件等）
5. 点击底部的**"提交"**按钮
6. 系统自动评分并给出反馈

### 3.4 正式考试

**操作路径：** 学员端 → 正式考试

#### 考试流程

```
查看考试列表 → 开始考试 → 逐题作答 → 交卷 → 查看成绩
```

#### 详细步骤

**第一步：查看可参加的考试**

- 考试列表显示：考试名称、考试时间、考试时长、状态
- 状态说明：
  - **未开始** — 考试时间未到，无法进入
  - **进行中** — 可以点击"进入考试"
  - **已结束** — 考试时间已过
  - **已完成** — 您已经交卷

**第二步：开始考试**

1. 点击"进入考试"按钮
2. 系统校验考试时间（开考前 15 分钟内可入场）
3. 考试开始后，页面顶部显示**倒计时**

**第三步：答题**

- 逐题选择答案，可随时切换上一题/下一题
- 已答和未答的题目有不同标记

**第四步：交卷**

- 点击"交卷"按钮提交全部答案
- 系统自动评分，立即显示成绩
- **注意：** 如果倒计时结束，系统会自动交卷

#### 考试规则

- 考试开始后**迟到 15 分钟**仍可入场
- 考试结束后系统允许 **5 分钟宽限时间**内交卷
- 超过宽限时间则判为**"已超时"**
- 及格线默认为 **60 分**（管理员可调整）
- 每道题的分值由试卷设定

### 3.5 成绩查询

**操作路径：** 学员端 → 成绩查询

- 显示所有已完成的考试记录
- 每条记录包含：考试名称、总分、是否通过、状态
- 成绩发布后才能查看（管理员未发布前看不到具体分数）

### 3.6 错题本

**操作路径：** 学员端 → 错题本

- 汇总所有练习中答错的题目
- 显示错题次数，便于重点复习
- 可重新作答错题

### 3.7 帮助中心

**操作路径：** 学员端 → 帮助

- 常见问题解答
- 操作指南
- 操作快捷键说明

## 4. 教师操作指南

教师登录后进入教师端，主要功能包括查看班级概况、管理考试安排和查看学员信息。

### 4.1 教师仪表盘

**操作路径：** 教师端 → 仪表盘

教师仪表盘展示：

- **学员总数** — 当前班级的学员人数
- **考试总数** — 已安排的考试数量
- **平均成绩** — 班级学员的平均考试成绩
- **近期考试** — 即将进行或正在进行的考试列表

### 4.2 考试管理

**操作路径：** 教师端 → 考试管理

- 查看所有考试安排的列表
- 创建新考试：填写考试名称、选择试卷、设置考试时间范围和时长
- 查看每场考试的参加人数和完成情况

### 4.3 学员管理

**操作路径：** 教师端 → 学员管理

- 查看所辖班级的全部学员
- 显示学员的姓名、邮箱、练习进度、考试完成情况
- 可点击查看单个学员的详细成绩

| 模块 | 说明 |
|------|------|
| 管理首页 | 系统数据概览 |
| 用户管理 | 管理系统用户 |
| 组织管理 | 管理培训机构 |
| 班级管理 | 管理教学班级 |
| 考试安排 | 创建和管理考试 |
| 试卷管理 | 创建和管理试卷 |
| 成绩管理 | 查看、复核、发布成绩 |
| 考试监控 | 实时查看考试进行状态 |
| 题库（练习） | 管理练习题库 |
| 题库（考试） | 管理考试题库 |
| 题库导入 | 批量导入题目 |
| 题库审核 | 审核待发布的题目 |

### 5.1 管理首页

**操作路径：** 管理端 → 管理首页

管理首页展示系统核心数据看板：

- **练习题数量** — 练习题库中的题目总数
- **考试题数量** — 考试题库中的题目总数
- **学员总数** — 注册学员人数
- **教师总数** — 注册教师人数
- **班级总数** — 已创建的班级数

### 5.2 用户管理

**操作路径：** 管理端 → 用户管理

#### 查看用户列表

- 显示所有系统用户的姓名、邮箱、角色、所属组织、创建时间
- 支持搜索筛选

#### 用户角色说明

| 角色 | 权限范围 |
|------|----------|
| 超级管理员 (super_admin) | 全部管理功能 |
| 学校管理员 (school_admin) | 本组织内的管理功能 |
| 教师 (teacher) | 查看班级学员、管理考试 |
| 题库编辑 (question_editor) | 创建和编辑题目 |
| 题库审核 (question_reviewer) | 审核题目 |
| 监考员 (invigilator) | 监控考试进行 |
| 学员 (student) | 练习、考试、查看成绩 |
| 审计员 (auditor) | 查看审计日志 |

### 5.3 组织管理

**操作路径：** 管理端 → 组织管理

- 查看所有培训机构列表
- 每个组织显示：名称、编号、联系方式、状态、班级数、学员数
- 可创建新的培训机构

### 5.4 班级管理

**操作路径：** 管理端 → 班级管理

- 查看所有班级列表
- 每个班级显示：班级名称、所属学校、学员人数
- 可创建新班级并关联到培训机构

### 5.5 考试安排

**操作路径：** 管理端 → 考试安排

#### 查看考试安排

- 显示所有考试安排的列表
- 每条记录包含：考试名称、试卷、班级、开始时间、结束时间、时长、状态

#### 创建考试安排

1. 点击"创建考试"
2. 填写以下信息：
   - **考试名称** — 如"2026年7月人工智能训练师五级认证考试"
   - **选择试卷** — 从已有试卷中选择
   - **选择班级** — 指定参加考试的班级
   - **开始时间** — 考试开始的日期和时间
   - **结束时间** — 考试结束的日期和时间
   - **考试时长** — 每位学员的答题时间（分钟）
   - **迟到入场时间** — 允许迟到的分钟数（默认15分钟）
3. 点击"保存"

#### 考试状态说明

| 状态 | 含义 |
|------|------|
| 未开始 | 当前时间早于考试开始时间 |
| 进行中 | 当前时间在考试时间范围内 |
| 已结束 | 当前时间晚于考试结束时间 |

### 5.6 试卷管理

**操作路径：** 管理端 → 试卷管理

#### 查看试卷列表

- 显示所有试卷
- 每份试卷显示：名称、题目数量、总分、创建时间

#### 创建试卷

1. 点击"创建试卷"
2. 填写试卷名称
3. 选择题目并设置每题分值
4. 系统自动计算总分
5. 点击"保存"

### 5.7 成绩管理

**操作路径：** 管理端 → 成绩管理

#### 查看成绩列表

- 显示所有学员的考试成绩
- 每条记录包含：学员姓名、考试名称、总分、是否通过、状态
- 支持按状态筛选：待复核 / 已发布

#### 成绩复核

1. 点击某条成绩的"复核"按钮
2. 查看逐题详情：每道题的学员答案、正确答案、得分情况
3. 可进行以下操作：
   - **调整分数** — 修改总分
   - **标记通过/不通过** — 手动调整通过状态
   - **添加复核备注** — 记录复核意见
4. 点击"保存复核结果"

#### 发布成绩

1. 选择需要发布的成绩（可多选）
2. 点击"发布成绩"按钮
3. 成绩发布后，学员即可在"成绩查询"中看到分数

#### 成绩状态说明

| 状态 | 含义 |
|------|------|
| 自动评分 (auto_graded) | 系统已自动评分，等待复核 |
| 已复核 (reviewed) | 管理员已复核 |
| 已发布 (published) | 成绩已发布，学员可查看 |
| 已超时 (expired) | 考试超时，系统自动判分 |

### 5.8 考试监控

**操作路径：** 管理端 → 考试监控

实时查看正在进行中的考试状态：

- **当前考试** — 考试名称、时间范围、时长
- **学员状态** — 每位学员的考试状态：
  - 未开始 — 尚未进入考试
  - 答题中 — 正在答题，显示已用时间
  - 已交卷 — 已提交试卷
  - 已超时 — 超过考试时间

### 5.9 数据报表

**操作路径：** 管理端 → 数据报表

报表页面提供多维度数据分析：

#### 成绩分布

- 按分数段统计人数（如 90-100、80-89、70-79 等）
- 直观展示成绩分布情况

#### 考试通过率

- 每场考试的参加人数、通过人数、通过率、平均分

#### 班级表现对比

- 各班级的学员人数、平均分、通过率

#### 薄弱环节分析

- 统计错题最多的题型和知识点
- 帮助教师了解学员的薄弱项，针对性加强教学

#### 近期活动

- 显示最近的考试和答题记录

#### 总体统计

- 考试场次、答题次数、通过人数、平均分、题库总量

### 5.10 系统设置

**操作路径：** 管理端 → 系统设置

管理员可配置以下系统参数：

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| 考试交卷宽限时间（秒） | 300 | 考试结束后允许继续交卷的秒数 |
| 考试迟到入场时间（分钟） | 15 | 考试开始后允许入场的分钟数 |
| 考试及格线（分） | 60 | 考试通过的最低分数 |
| 练习模式显示答案 | 开启 | 练习时是否立即显示正确答案 |
| 练习最大尝试次数 | 3 | 每道练习题最多可尝试次数（0 = 不限） |
| 密码最小长度 | 8 | 用户密码的最小长度要求 |
| 会话超时时间（分钟） | 120 | 用户无操作后自动登出的时间 |

#### 修改设置

1. 修改对应参数的值
2. 点击"保存设置"
3. 设置立即生效

### 5.11 审计日志

**操作路径：** 管理端 → 审计日志

记录系统中的关键操作，便于追溯和审查。

#### 支持的筛选条件

- **操作类型** — 如登录、创建考试、提交试卷、发布成绩等
- **操作人** — 按用户筛选
- **时间范围** — 按日期筛选

#### 日志内容

每条日志包含：操作人、角色、操作类型、操作对象、详细信息、操作时间。

### 5.12 媒体素材工坊

**操作路径：** 管理端 → 媒体素材工坊

管理员可使用 AI 生成教学素材。

#### AI 图片生成

1. 在"图片生成"标签页中输入描述文字（如"一张包含多辆汽车的街道照片"）
2. 点击"生成图片"
3. 等待 AI 生成完成后，可预览和下载图片
4. 生成的图片可用于实操题目中的图片标注、图片清洗等任务

#### TTS 音频生成

1. 在"音频生成"标签页中输入需要转写的文字
2. 选择语音类型
3. 点击"生成音频"
4. 可预听和下载音频文件
5. 生成的音频可用于音频转写练习题

### 5.13 题库管理

系统题库分为**练习题库**和**考试题库**两部分。

#### 练习题库

**操作路径：** 管理端 → 题库（练习）

- 管理学员日常练习使用的题目
- 支持新增、编辑、删除题目
- 题型包括：单选题、判断题

#### 考试题库

**操作路径：** 管理端 → 题库（考试）

- 管理正式考试使用的题目
- 支持新增、编辑、删除题目
- 考试题独立于练习题，确保考试内容不泄露

#### 题库导入

**操作路径：** 管理端 → 题库导入

- 支持 Word 文档 (.docx) 批量导入题目
- 导入格式：每题包含题干、选项、正确答案
- 导入后题目自动进入草稿状态

#### 题库审核

**操作路径：** 管理端 → 题库审核

- 题库编辑员创建的题目需要审核员审核
- 审核通过后题目才会进入正式题库
- 审核员可查看题目详情、批准或驳回

## 6. 常见问题

### Q: 忘记密码怎么办？

请联系系统管理员重置密码。管理员可在"用户管理"中操作。

### Q: 考试中途断网了怎么办？

如果网络恢复后考试时间未结束，重新登录系统进入考试即可继续答题。系统会自动恢复您之前的答题进度。如果考试时间已结束，系统会按已提交的答案自动评分。

### Q: 考试时间到了但没有点"交卷"会怎样？

系统会在倒计时结束后自动交卷，已选择的答案会被提交评分。未作答的题目记为零分。

### Q: 为什么看不到考试成绩？

成绩需要管理员**发布**后学员才能查看。如果考试结束后看不到成绩，可能是管理员尚未发布，请耐心等待或联系管理员。

### Q: 练习题答错了怎么复习？

答错的题目会自动进入"错题本"。在学员端点击"错题本"即可查看所有错题并重新作答。

### Q: 及格分数可以修改吗？

可以。管理员在"系统设置"中修改"考试及格线"参数即可。修改后对新提交的考试立即生效。

### Q: 一个学员可以参加多次同一场考试吗？

默认情况下，每位学员对每场考试只有一次答题机会。交卷后无法重新参加同一场考试。

### Q: 如何批量创建学员账号？

管理员可在"用户管理"中逐个创建，或通过数据库种子脚本批量导入。如需大批量导入，请联系技术人员处理。

### Q: 考试题目可以修改吗？

- **考试进行中** — 绝对不允许修改试卷和题目
- **考试结束后** — 可以修改题库中的题目内容，但已提交的考试成绩不会受影响
- 建议在创建考试安排前确认试卷内容无误

### Q: 系统支持哪些浏览器？

推荐使用 Chrome、Edge 或 Firefox 最新版本浏览器。为获得最佳体验，建议屏幕分辨率不低于 1366×768。
